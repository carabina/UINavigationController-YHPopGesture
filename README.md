UINavigationController+YHPopGesture
======================

A pop gesture in viewcontroller even if there is a horizontal scrollview.


Installation
------------

Use [CocoaPods](http://cocoapods.org).

```ruby
pod 'UINavigationController+YHPopGesture'
```
